git stash

基本使用

某些场景：
正改着代码，然后有个bug.

  • 使用stash保存现在的代码 –> 新代码没有提交；工作区恢复为原状。
  • 修改bug,并提交新代码。
  • 恢复stash的代码。

问题解决

如果新修改的代码 跟 stash 修改的代码有冲突。
git会提示先提交最新的修改。再执行stash, 会实现代码合并。

vim 查找

用vim打开文件，需要搜索某个字符串。

  • :/str enter –> 输入
  • 按 n 向下搜索。 按 N 向上搜索。
  • :/^str enter –> 搜索行首字符串
  • :/str$ enter –> 搜索行尾字符串
